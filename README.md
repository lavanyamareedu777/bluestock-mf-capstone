# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This project analyzes mutual fund industry data using Python, SQLite, Jupyter Notebook, and Power BI. The project covers ETL development, exploratory data analysis, performance analytics, advanced risk analytics, and dashboard creation.

## Objectives

* Build an ETL pipeline for mutual fund datasets
* Clean and validate data
* Store processed data in SQLite
* Perform EDA and visualization
* Compute performance metrics (CAGR, Sharpe, Alpha, Beta, Drawdown)
* Conduct advanced analytics (VaR, CVaR, Cohort Analysis, Recommender)
* Develop an interactive Power BI dashboard
